## Code of Conduct

By participating in this project, you agree to abide by the Code of Conduct and respect the original authorship and branding.

## How to Contribute

We welcome contributions in the form of:
-   **Bug Reports**: Please open an issue if you encounter any problems.
-   **Feature Requests**: Have an idea? Let us know!
-   **Pull Requests**: Submit your improvements via PRs.

### Guidelines for Contributions

1.  **Fork the Repository**: Create your own copy of the project.
2.  **Make Changes**: Improve the code, fix bugs, or add features.
3.  **Respect Authorship**: Do not remove or alter the original credits, branding, or copyright notices (e.g., "Developed by 𝕏0L0™").
4.  **Submit a Pull Request**: Provide a clear description of your changes.
